# sales-dashboard-analysis
Interactive sales analysis dashboard project using Excel, SQL and Power BI

## Project Overview
This project analyzes factors affecting profitability using Power BI.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel

## Business Problem
Identify factors impacting profit across regions, categories, and discounts.

## Dashboard Features
- KPI Cards
- Regional Profit Analysis
- Monthly Sales Trend
- Discount vs Profit Analysis
- Sub-Category Profitability

## Key Insights
- West region generated highest profit
- Higher discounts impacted profitability negatively
- September and November had peak sales
- Binders and Copiers were top profit contributors
